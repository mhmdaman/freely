# Freely — Material Linen Canvas Music Player

Freely is a high-fidelity, tactile music streaming application built with React and the YouTube Iframe API. It features a unique **Material Linen Canvas** aesthetic, blending the warmth of a vintage vinyl record shop with the intimate feel of a handcrafted leather journal.

## 🎨 Aesthetic: Material Linen Canvas

Freely moves away from cold, digital interfaces toward a warm, analog-inspired design system:

- **Raw Linen & Warm Ecru**: The primary surfaces are textured with a visible linen weave grain, creating a physical sense of presence.
- **Polaroid Album Art**: Every track is presented as a squared polaroid frame with a thick ecru border, featuring a slight 1.5° tilt and a rhythmic "heartbeat" pulse during playback.
- **Tailor's Ruler Progress**: The progress bar is styled as a vintage measuring tape, featuring a weathered brass "snap button" thumb.
- **Leather Medallion Controls**: The play button is an embossed terracotta leather medallion with custom stitching details.
- **Brush Stroke Visualizer**: Soft, fluid linen-colored brush strokes rise from the canvas, mimicking the motion of pulled threads.
- **Catalog Card Library**: Songs and search results appear as library index cards with dashed tailoring pattern dividers.

## 🛠 Features

- **YouTube Integration**: Search and stream any track from the vast YouTube Music library.
- **Immersive Journal**: Save your favorite records to your personal "Journal" (Library).
- **Recent Logs**: Automatically tracks your recently flipped records for quick access.
- **Curated Collections**: Browse mood-based collections styled as boutique shop categories.
- **Tactile Transitions**: Smooth, warm vignettes and amber shimmers provide a premium, sunlit loading experience.

## 🚀 Getting Started

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local development server.
4. Open the link in your browser and "Open the Journal" to begin.

## 📜 Technology Stack

- **Framework**: React (via CDN/Babel)
- **Styling**: Tailwind CSS & Vanilla CSS for custom textures
- **API**: YouTube Data API v3 & YouTube Iframe API
- **Design**: Custom SVG filters for linen and leather textures
